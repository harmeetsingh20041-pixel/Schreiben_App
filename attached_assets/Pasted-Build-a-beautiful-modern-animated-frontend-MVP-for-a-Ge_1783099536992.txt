Build a beautiful, modern, animated frontend MVP for a German writing correction tool.

This is a new standalone app for my personal German teaching workflow and Physics Wallah-related student practice. The app is mainly for A1/A2 German learners.

Working app name:
German Writing Coach

Main purpose:
Students should be able to log in, choose a predefined German writing question, paste their answer, and receive accurate line-by-line correction.

The most important requirement:
The correction system must not overcorrect.

If a student writes a simple but correct A1/A2 sentence, the app should show that the sentence is correct. It should not rewrite it into advanced German. It should only correct real mistakes such as grammar, articles, case, verb position, conjugation, spelling, tense, prepositions, or unclear meaning.

Use this mock feedback:

Line 1:
Original: “Ich habe meinen Mutter ein Geschenk gegeben.”
Corrected: “Ich habe meiner Mutter ein Geschenk gegeben.”
Status: Small grammar issue
Changed: “meinen” → “meiner”
Explanation: “After ‘geben’, the person receiving something is usually in Dativ. ‘Die Mutter’ becomes ‘meiner Mutter’.”
Grammar topic: Dativ

Line 2:
Original: “Ich spiele gern Badminton im Park.”
Corrected: “Ich spiele gern Badminton im Park.”
Status: Correct
Explanation: “Correct. No correction needed.”
Grammar topic: Word order

Line 3:
Original: “Gestern ich bin ins Kino gegangen.”
Corrected: “Gestern bin ich ins Kino gegangen.”
Status: Small grammar issue
Changed: “ich bin” → “bin ich”
Explanation: “When the sentence starts with ‘Gestern’, the verb comes in position 2.”
Grammar topic: Verb position

Line 4:
Original: “Ich habe gehen Schule.”
Corrected: “Ich bin zur Schule gegangen.”
Status: Major issue
Explanation: “The original sentence is not grammatically clear. For movement in the past, use ‘bin ... gegangen’.”
Grammar topic: Perfekt
